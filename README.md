# 사주 로또 추천기

과거 로또 6/45 당첨 번호 통계와 간이 사주 오행 점수를 함께 반영해 추천 조합을 만드는 브라우저 앱입니다.

## 실행

`index.html`을 브라우저에서 열면 바로 사용할 수 있습니다.

## 데이터 갱신

새 회차가 나온 뒤 아래 명령을 실행하면 `data/lotto-results.json`과 `data/lotto-results.js`가 갱신됩니다.

```powershell
node tools/update-lotto-data.mjs
```

데이터 수집은 동행복권 JSON 엔드포인트를 먼저 시도하고, 접근이 막히면 공개 회차 목록 페이지를 보조 소스로 사용합니다.

## 주의

로또 추첨은 독립 난수 이벤트이므로 과거 통계, 사주, 추천 점수는 1등 확률을 실제로 높인다고 볼 수 없습니다. 이 앱의 추천 점수는 번호 조합을 고르는 참고 지표입니다.

## 명리 모델

명리 반영 방식은 [docs/research-notes.md](docs/research-notes.md)에 정리했습니다. 현재 버전은 천간·지지·장간·월령·십성·용신 후보를 계산하고, 그 결과를 투명한 번호-오행 매핑으로 변환합니다.

## 위치와 개운 루틴

사는 곳이나 현재 위치를 반영해 사주상 보완 방향의 판매점 검색 링크, 추천 구매 시간대, 행운 색상·의상·음식·소품을 함께 제안합니다. 실제 판매점명과 영업 여부는 지도와 동행복권 판매점 조회에서 최신 정보를 확인하는 구조입니다.

## 90점 이상 추천

기본 추천점수 필터는 90점입니다. 앱은 더 많은 후보를 만들고 상위 후보를 다시 개선해서, 통계 신호와 품질 관문을 함께 통과한 조합 위주로 보여줍니다. 점수는 당첨 확률이 아니라 앱 내부의 추천 품질 등급입니다.

`가장 높은 점수부터 보기`를 켜면 추천 세트 개수만큼 최고점 후보를 점수순으로 보여줍니다. 끄면 최고점 후보 중에서도 서로 너무 비슷한 조합을 줄여 다양성을 확보합니다.

## 복권집 TOP 3

구매 루틴에는 오늘/내일 구매운과 추천 복권집 TOP 3가 표시됩니다. 복권집 추천은 입력한 생활권, 사주상 보완 오행, 재성 오행, 방향성, 동행복권 당첨 판매점 공개 목록 등장 여부를 함께 반영합니다.

## 배포

무료 배포는 링크 배포 + PWA 설치를 추천합니다. 자세한 방법과 APK 배포 비교는 [docs/deployment.md](docs/deployment.md)에 정리했습니다.
